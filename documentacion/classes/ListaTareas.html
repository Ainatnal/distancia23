Clase para manejar una lista de tareas.

La siguiente clase sirve para gestionar lista de tareas pendientes. El primer método de la clase sirve para agregar una tarea a la lista. El segundo método de la función, añadido en la versión 2.0, sirve para imprimir la lista de tareas.
